# Fake News Detection using NLP & Machine Learning

This project uses TF-IDF vectorization and Logistic Regression to classify news as Real or Fake.  
Dataset: Fake and Real News Dataset (Kaggle via kagglehub)

## Features
- Text cleaning & preprocessing
- TF-IDF vectorization
- Logistic Regression model
- WordCloud visualizations
- Confusion Matrix
- 98% accuracy

## Files Included
- FakeNewsDetection.ipynb
- main.py
- visuals/ (WordCloud + Confusion Matrix)
- models/ (model.pkl, vectorizer.pkl)
- requirements.txt

## How to Run
```python
pip install -r requirements.txt
python main.py
